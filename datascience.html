Data Science

Posted by Neil Seward on May 1, 2015

In between school years, I decided to try out the Data Science specialization offered by John Hopkin's through Coursera. It is a great introductory course offering for someone that wants to get their feet wet in data analytics.

I took 5 courses over the summer. Each course had unique project that I worked on that introduced me to the R language and the power it has in analytics.

Getting And Cleaning Data

This course came right after the introductory R course and was the first real application of r in data analysis. The course project was to get specific means of body activities from a phone accelerometer data.

Read the accelerometer data and merge training and test sets.
                        
trainData <- read.table("data/train/X_train.txt")
trainLabel <- read.table("data/train/y_train.txt")
trainSubject <- read.table("data/train/subject_train.txt")
testData <- read.table("data/test/X_test.txt")
testLabel <- read.table("data/test/y_test.txt")
testSubject <- read.table("data/test/subject_test.txt")

data <- rbind(trainData, testData)
label <- rbind(trainLabel, testLabel)
subject <- rbind(trainSubject, testSubject)
Extract all mean and std measurement names and label the end dataset. Read different kinds of activities recorded on the phone and insert as first column. 
                        
features <- read.table("data/features.txt")
featureLabels <-  grep("-(mean|std)\\(\\)", features[, 2])
data <- data[,featureLabels]

names(data) <- features[featureLabels,2]
  
activity <- read.table("data/activity_labels.txt")
label[, 1] <- activity[label[,1],2]
Bind the subject and activity to the existing data and calculate the average of every subset of each activity accelerometer feature that includes std or mean. Write the data set to file 
                        
names(label) <- "activity"
names(subject) <- "subject"
  
dataSet <- cbind(data,label,subject)
averageData <- ddply(dataSet, .(subject,activity),function(x) colMeans(x[,1:66]))

write.table(averageData,"averageData.txt",row.name=FALSE)

Exploratory Data Analysis

This course taught what can be represented with the data. The asignment shown involved creating several different representations of emmisions data to anwer basic questions on whether emission were increasing in different areas.

Question: Have total emissions from PM2.5 decreased from 1999 to 2008?

Read data, aggregate emission sums by year, and plot a histogram of the sums. 
                        
NEI <- readRDS("summarySCC_PM25.rds")
    
emissionsPerYear <- aggregate(NEI$Emissions,by=list(NEI$year),FUN=sum)
  
png(filename = "plot1.png", width=480,height=480)
plot(emissionsPerYear$Group.1, emissionsPerYear$x, xlab="Year", ylab="Total Emissions(tons)", 
main = "Emissions By Year",type='l')
dev.off()

Question: Have total emissions from PM2.5 decreased from 1999 to 2008 in Baltimore?

Read data, subset to Baltimore area zip code, aggregate sums and plot histogram. 
                        
NEI <- readRDS("summarySCC_PM25.rds")
    
Baltimore <- subset(NEI,NEI$fips=="24510")    
emissionsPerYear <- aggregate(Baltimore$Emissions,by=list(Baltimore$year),FUN=sum)
  
png(filename = "plot2.png", width=480,height=480)
plot(emissionsPerYear$Group.1, emissionsPerYear$x, xlab="Year", ylab="Total Emissions(tons)",main = "Baltimore Emissions By Year", type='l')
dev.off()

Question: Have four types of emission sources from PM2.5 decreased from 1999 to 2008 in Baltimore?

Read data, subset to Baltimore area zip code, aggregate each year in each type and plot four histograms using ggplot package. 
                        
NEI <- readRDS("summarySCC_PM25.rds")
    
Baltimore <- subset(NEI,NEI$fips=="24510")    
emissionsPerYearPerType <- aggregate(Emissions ~ year, Baltimore,sum)
  
png(filename = "plot3.png", width=480,height=480)
plot <- ggplot(Baltimore,aes(factor(year),Emissions,fill=type)) +
geom_bar(stat="identity") +
theme_bw() + guides(fill=FALSE)+
facet_grid(.~type,scales = "free",space="free") + 
labs(x="year", y=expression("Total PM"[2.5]*" Emission In Tonnes")) + 
labs(title=expression("PM"[2.5]*" Emissions, Baltimore City 1999-2008 by Source Type"))
  
print(plot)
dev.off()

Question: What are the emissions trends in the US from 99-08 for coal combustion?

Read data, subset to only contain coal and combustion data using grepl, plot by factor using ggplot package. 
                        
NEI <- readRDS("summarySCC_PM25.rds")
SCC <- readRDS("Source_Classification_Code.rds")
    
allCombustion <- grepl("comb", SCC$SCC.Level.One, ignore.case=TRUE)
allCoal <- grepl("coal", SCC$SCC.Level.Four, ignore.case=TRUE)
coalCombustion <- (allCoal & allCombustion)

combustionSCC <- SCC[coalCombustion,]$SCC
combustionNEI <- NEI[NEI$SCC %in% combustionSCC,]
  
png("plot4.png",width=480,height=480)
plot <- ggplot(combustionNEI,aes(factor(year),Emissions/10^5)) +
geom_bar(stat="identity",fill="grey",width=0.75) +
theme_bw() +  guides(fill=FALSE) +
labs(x="year", y=expression("Total PM"[2.5]*" Emission (10^5 Tons)")) + 
labs(title=expression("PM"[2.5]*" Coal Combustion Emissions In The US from 99-08"))
  
print(plot)
dev.off()

Question: What are the vehicle emission trends in Baltimore and LA from 99-08?

Read data, subset to only contain vehicle data using grepl and in Baltimore and Los Angeles zip, plot by year factor using ggplot package. 
                        
NEI <- readRDS("summarySCC_PM25.rds")
SCC <- readRDS("Source_Classification_Code.rds")
    
vehicle <- grepl("vehicle", SCC$SCC.Level.Two, ignore.case=TRUE)
vehicleSCC <- SCC[vehicle,]$SCC
vehicleNEI <- NEI[NEI$SCC %in% vehicleSCC,]

Baltimore <- subset(vehicleNEI,vehicleNEI$fips=="24510")
LosAngeles <- subset(vehicleNEI,vehicleNEI$fips=="06037")
cities <- rbind(Baltimore,LosAngeles)
  
png("plot6.png",width=480,height=480)
plot <- ggplot(cities,aes(x=factor(year), y=Emissions, fill=city)) 
+ geom_bar(aes(fill=year),stat="identity") +
facet_grid(scales="free", space="free", .~city) + theme_bw() +  
guides(fill=FALSE) + labs(x="year", y=expression("Total PM"[2.5]*" Emission (10^5 Tons)")) + 
labs(title=expression("PM"[2.5]*" Vehicle Emissions in Baltimore and Los Angeles From 99-08"))
  
print(plot)
dev.off()
